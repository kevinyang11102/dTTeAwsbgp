# 停车场管理系统 java523

## 项目概述

停车场管理系统是一个基于B/S架构和MVC模式的Web应用。该系统采用前后端分离的设计，前端使用Vue.js框架，后端采用SpringBoot框架，数据库使用MySQL，旨在为用户提供便捷的停车服务与管理功能。

## 技术栈

- **架构**: B/S、MVC
- **系统环境**: Windows
- **开发环境**: 
  - IDEA
  - JDK 1.8
  - Maven
  - MySQL
- **技术栈**: 
  - Java
  - MySQL
  - SpringBoot
  - Mybatis-Plus
  - Vue.js

## 功能模块

### 普通用户

- 个人信息管理
- 预定停车位
- 缴费信息管理

### 管理员用户

- 用户信息管理
- 车辆信息管理
- 停泊车辆管理
- 车辆进出管理

## 系统角色

- 普通用户：具有基本的个人信息管理以及停车相关操作权限。
- 管理员：具备管理用户信息、车辆信息以及停车场日常运作的权限。

## 运行环境

- **服务器**: 推荐使用符合Java应用运行要求的Windows服务器
- **数据库**: MySQL服务器

## 部署步骤

1. 环境搭建：准备IDEA、JDK 1.8、Maven、MySQL。
2. 数据库配置：根据提供的文档，设置MySQL数据库。
3. 后端部署：使用IDEA或Maven命令行编译项目，部署至Java Web服务器。
4. 前端部署：将Vue项目打包，部署至Web服务器对应目录。
